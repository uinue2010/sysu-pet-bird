# sysu-pet-bird

这是一个原生 macOS 菜单栏应用原型：桌面显示透明悬浮宠物窗口，菜单栏可切换表情、调整大小/透明度、置顶、重新扫描素材，并包含最基础的自动切换规则。

## 运行

```bash
swift run ZhongDaBirdPet
```

也可以生成一个 `.app`：

```bash
bash Scripts/build_app.sh
open ".build/中大鸟桌宠.app"
```

默认扫描素材目录：

```text
~/Documents/中大鸟
```

打包成 `.app` 时，项目里的 `Assets/ZhongDaBird` 也会一起放进应用包。这样即使本机没有 `~/Documents/中大鸟`，应用也能使用内置素材。

## 已实现

- 透明、无边框、可拖动的桌面宠物窗口
- 菜单栏入口
- 手动切换扫描到的表情
- 暂停/恢复自动切换
- 重新扫描素材
- 调整大小、透明度、置顶
- 用 ImageIO 识别真实图片格式，而不是相信扩展名
- 后缀不准时，自动复制到 `Application Support/ZhongDaBirdPet/Normalized` 并改成正确扩展名再加载
- 自动规则：
  - 11:30-12:30 和 17:00-18:00 优先切换到“饿了”
  - 21:30 之后优先切换到“晚安”
  - 其他时间约每 2 分钟随机切换一个表情
  - 随机时会避开“饿了/晚安”，并尽量避免连续抽到当前同一个表情

## 目前素材检查结果

当前 `~/Documents/中大鸟` 下扫描到 19 个图片文件。多个 `.jpeg` 的真实格式是 GIF，包括“沉迷科研 / 晚安 / 听音乐 / 开心 / 学习ing / 比心 / 撒花 / 加油 / 疑惑 / 饿了 / 锦鲤附体”等，应用会自动按 GIF 处理。另有一个无扩展名的 `休息一下_png序列`，真实格式是 PNG，也会被自动复制成可加载的 `.png`。
